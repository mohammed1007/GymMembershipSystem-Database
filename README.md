# Gym Membership Database System
A fully normalized (1NF-3NF) relational database designed for a gym membership system, including spa operations and class scheduling.

## Phase 1 Deliverables
- **Conceptual ERD:** Chen Notation (Entity-Relationship Diagram).
- **Logical Schema:** Relational Mapping with Foreign Key constraints.
- **Physical Implementation:** SQL Server (T-SQL) scripts.

## How to Run
1. Open `schema.sql` in SQL Server Management Studio.
2. Execute the script to generate the database structure.
